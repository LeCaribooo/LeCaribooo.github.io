Pour la peine quelque commande linux qui pourrait vous être utile
- cat nomdufichier pour afficher le contenu d'un fichier sur votre terminal
- ls pour voir la liste des fichiers dans le dossier ou tu te trouves
- cd pour te déplacer entre les fichiers (cd nomdufichier)
- whois pour voir des informations sur le propriétaire d'un nom de domaine :eyes:
Pour rappelle le nom de domaine c'est de la "forme" https://tonnomdedomain.com
